サンプル記事：オペアンプの基礎

2025.12.06 Electronics

この記事は、MathJaxによる数式レンダリングとHighlight.jsによるシンタックスハイライトのデモを兼ねたサンプル記事です。

1. オペアンプとは

オペアンプ（Operational Amplifier）は、非常に高い増幅率を持つ差動増幅器です。 理想的なオペアンプは以下の特性を持ちます：

  • 無限大の開ループゲイン: $A_v \to \infty$
  • 無限大の入力インピーダンス: $Z_{in} \to \infty$
  • ゼロの出力インピーダンス: $Z_{out} = 0$
  • 無限大の帯域幅

2. 反転増幅回路

最も基本的なオペアンプ回路である反転増幅回路を考えます。 入力電圧 $V_{in}$ と出力電圧 $V_{out}$ の関係は次式で表されます：

$$ V_{out} = -\frac{R_f}{R_{in}} V_{in} $$

ここで、$R_f$ はフィードバック抵抗、$R_{in}$ は入力抵抗です。 ゲイン $A$ は次のように定義されます：

$$ A = -\frac{R_f}{R_{in}} $$

3. コード例

以下は、オペアンプ回路のゲインを計算するPythonコードです： 

def calculate_inverting_gain(r_feedback: float, r_input: float) -> float:
    """
    反転増幅回路のゲインを計算する
    
    Args:
        r_feedback: フィードバック抵抗 [Ω]
        r_input: 入力抵抗 [Ω]
    
    Returns:
        ゲイン（負の値）
    """
    return -r_feedback / r_input


# 使用例
R_f = 10e3  # 10 kΩ
R_in = 1e3  # 1 kΩ

gain = calculate_inverting_gain(R_f, R_in)
print(f"ゲイン: {gain:.1f}")  # ゲイン: -10.0

3.1 C++での実装

同様の計算をC++で実装すると以下のようになります： 

#include <iostream>
#include <cmath>

class OpAmpCircuit {
private:
    double r_feedback;
    double r_input;

public:
    OpAmpCircuit(double rf, double rin) 
        : r_feedback(rf), r_input(rin) {}

    double calculate_gain() const {
        return -r_feedback / r_input;
    }

    double calculate_output(double v_in) const {
        return calculate_gain() * v_in;
    }
};

int main() {
    OpAmpCircuit circuit(10e3, 1e3);  // Rf = 10kΩ, Rin = 1kΩ
    
    std::cout << "Gain: " << circuit.calculate_gain() << std::endl;
    std::cout << "Vout (Vin=1V): " << circuit.calculate_output(1.0) << " V" << std::endl;
    
    return 0;
}

4. 伝達関数

実際のオペアンプには有限の帯域幅があり、周波数特性を考慮する必要があります。 単極モデルを仮定すると、開ループ伝達関数は：

$$ A(s) = \frac{A_0}{1 + s/\omega_p} $$

ここで、$A_0$ はDCゲイン、$\omega_p$ は支配極の角周波数です。 ゲイン帯域幅積（GBW）は $A_0 \omega_p$ で与えられます。
